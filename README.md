## ğŸš€ Getting Started

### Prerequisites

- Qt 5.15 or Qt 6 (with MinGW or MSVC compiler)
- Qt Creator (recommended) or VS Code with Qt extension

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/r1ckshot/FitnessCenterQt.git
   cd FitnessCenterQt
   ```

2. **Open the project**
   - Open `Silownia.pro` in Qt Creator
   - Or use command line with qmake

3. **Build and run**
   - In Qt Creator: Press `Ctrl+R` (or `Cmd+R` on macOS)
   - Or via command line:
     ```bash
     qmake Silownia.pro
     make
     ./Silownia  # or Silownia.exe on Windows
     ```
